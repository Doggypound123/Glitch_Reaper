# Glitch_Reaper
Glitch_Reaper is a simple lightweight Chrome extension built to execute diabolical bugs before they run into the void. It husses down recent user actions, page details, browser information, and hidden errors. Every captured detail is forged into a clean, structured bug report that developers can investigate immediately. When the system breaks, Glitch_Reaper already knows what happened.
